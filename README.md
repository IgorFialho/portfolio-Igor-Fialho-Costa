## Portfólio — Igor Fialho Costa

Este projeto é uma página de currículo/portfólio em formato de site estático para o desenvolvedor **Igor Fialho Costa**.

### Arquivos principais

- `index.html` — marcação HTML da página (estrutura, seções e conteúdo).
- `styles.css` — estilos do layout (tema escuro, responsivo, foco em tipografia e acessibilidade).
- `script.js` — comportamento básico (menu mobile, fundo do header ao rolar, ano automático no rodapé).

### Como rodar

1. Abra esta pasta (`curriculo`) no seu editor ou explorador de arquivos.
2. Clique duas vezes em `index.html` para abrir no navegador **ou**:
3. Use uma extensão de live server (como o Live Server do VS Code/Cursor) e sirva o arquivo `index.html`.

Não há dependências de build; é um site HTML/CSS/JS puro.

### Ajustes recomendados

- Atualize o e-mail em `index.html` na seção de **Contato** (`mailto:`) para o seu endereço real.
- Se quiser publicar, você pode usar GitHub Pages, Vercel ou outro provedor estático apontando para esta pasta.
